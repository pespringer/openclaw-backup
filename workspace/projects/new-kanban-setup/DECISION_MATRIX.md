# Weighted Decision Matrix for Evaluating Project Ideas

## Overview
This weighted decision matrix provides a structured framework for evaluating project ideas based on various criteria. Each criterion is assigned a weight to reflect its importance. Scores for each idea are calculated as the weighted sum of points across all criteria.

---

## Criteria
Below are the evaluation criteria and their respective weights:

| Criterion            | Weight (%) |
|----------------------|------------|
| Feasibility          | 25         |
| Market Potential     | 30         |
| Time to Implement    | 20         |
| Cost to Implement    | 15         |
| Innovation Factor    | 10         |

### Explanation
- **Feasibility** refers to how practical the idea is to implement given available resources.
- **Market Potential** examines demand and scalability in the target market.
- **Time to Implement** judges how quickly the idea can be brought to market.
- **Cost to Implement** evaluates the financial resources required.
- **Innovation Factor** measures how unique and innovative the idea is.
