# STORY-002 — Evaluate top 3 ideas against scoring matrix

## Status
Backlog




## Story
As a founder/operator, I want to evaluate the top 3 project ideas against the scoring matrix, so that I can select the strongest option with confidence.

## Why this matters
- Prevents subjective decision-making.
- Makes the first product choice defensible.
- Keeps execution focused on the highest-value path.

## Acceptance Criteria
- [ ] Top 3 candidate ideas are listed.
- [ ] Each idea is scored against the weighted matrix.
- [ ] Scores and rationale are documented clearly.
- [ ] A recommended winner is identified.

## Projected Work
- Estimate: M
- Approach:
  1. Pull the top candidate ideas.
  2. Score each against the decision matrix.
  3. Compare total scores and risks.
  4. Recommend the best first product.
- Dependencies: DECISION_MATRIX.md
- Risks: Bias toward technically interesting ideas instead of most sellable ones.
